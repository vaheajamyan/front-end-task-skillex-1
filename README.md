# Skillex
## Task: Products List with filters

## Objective
Implement a dynamic product filtering system using **React.js**.

### Main Requirements
- Fetch and display products using the Products API (refer to the README.md file in the products directory for API documentation).
- Display available filter options.
- Allow users to apply multiple filters simultaneously.
- Implement pagination.

### Testing (Optional)
- Implement unit tests to check the functionality of the filtering system using a testing library such as **Jest** or **React Testing Library**.

## Stack
- **React.js**
- **CSS or SCSS**

## Expected Deliverables
- A fully functional, responsive, and user-friendly product listing page with filtering and pagination.
